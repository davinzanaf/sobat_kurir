# Sobat Kurir

Sobat Kurir adalah aplikasi jasa pengiriman barang berbasis PHP Native dan MySQL.

## Fitur
- Login Admin
- Kelola Kurir
- Kelola Tarif
- Data Pesanan
- Cek Ongkir
- Buat Pesanan
- Tracking Pesanan
- Dashboard Kurir
- Update Status Pengiriman

## Role Pengguna
- Admin
- Kurir
- Customer

## Teknologi
- PHP Native
- MySQL / MariaDB
- HTML
- CSS
- Laragon
- HeidiSQL

## Cara Menjalankan
1. Simpan project di folder `www` Laragon
2. Buat database `db_sobat_kurir`
3. Import tabel yang dibutuhkan
4. Atur koneksi database di `config/koneksi.php`
5. Jalankan di browser melalui localhost
